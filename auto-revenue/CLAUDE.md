# 自動収益システム 運用ルール

## 世界観・コンセプト

AIツールを活用したビジネス自動化・副業ノウハウ・生産性向上テクニックに特化したアカウント。

- **オーナーは判断するだけ。手は動かさない。**
- すべての投稿・収益化は自動化が前提
- 人手が必要な初期設定は「1回だけの判断」として明示する

## ターゲット

- メインターゲット: 20代〜40代のビジネスパーソン・個人事業主・副業で効率的に稼ぎたい人
- 訴求ポイント: 「時間をかけずに成果を出したい」「AIを使って仕事を自動化したい」「今日から使えるノウハウが欲しい」
- 主な流入経路: X（Twitter）/ note / SEO

## 口調・投稿方針

- 丁寧で信頼性の高いトーン（馴れ馴れしくせず、でも堅すぎない）
- 抽象論ではなく、今日から使える実用的な内容を提供する
  - ステップ（手順を番号で明示）
  - ツール名（具体的なサービス・アプリ名）
  - 数値（時間・コスト・成果の数字）
- 1投稿＝1テーマ。詰め込みすぎない

## 収益化モデル

- アフィリエイト（A8.net / Amazon Associates / 楽天）
- 案件詳細は `affiliates/` を参照

## 投稿ルール

- ネタ元は `sources/` に登録済みのリストのみ使用する
- 生成した投稿は必ず `posts/` に保存してから公開する
- スケジュールは `schedule.md` で管理する
- アフィリエイトリンクなしの外部リンクは投稿に含めない

## アフィリエイトリンクの絶対ルール

- すべての遷移先は商品・サービスの個別ページ（トップページ遷移は収益ゼロ）
- リンクには必ずアフィリエイトタグを付与すること
- 利益が発生しないリンクを本番に出すことは禁止

## 禁止事項

- ランダム生成・推定値のコンテンツ
- 手動投稿・手動申請を前提とするフロー
- アフィリエイトタグなしの外部リンク
